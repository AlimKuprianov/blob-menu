## Requirements

- iOS 13.0  
- Xcode 11.4    
- Swift 5.0


## Installation
You can install `blob-menu` in several ways:

- Add source files to your project.

<br>

- Use [CocoaPods](https://cocoapods.org):
``` ruby
pod 'blob-menu'
```

<br>

- Use [Carthage](https://github.com/Carthage/Carthage):
```
github "Ramotion/blob-menu"
```
